Start a discovery session
Select a scenario from the sidebar or start fresh. The AI will conduct an adaptive discovery conversation the way a senior SE would.
// Gap Analysis
Complete discovery first, then run gap analysis to identify security gaps scored against CIS Controls, NIST CSF, SOC2, and ISO27001.
// Vendor Recommendations
Requires gap analysis. Vendor recommendations are grounded in the gap analysis — Claude reasons over real vendor capability data, not marketing claims.
// Objection & Constraint Handling
Detects constraints from the discovery transcript (existing investments, budget, team size, contracts, migration risk) and generates SE responses grounded in real enterprise playbooks.
// Architecture Options
Generates 4 alternative architecture paths: Best Security / Best Cost / Fastest Time to Value / Lowest Operational Overhead. Each with vendors, risks, and when to choose it.
// Stakeholder Alignment
Generates stakeholder-specific messaging for CISO, CIO, Security Architect, IAM team, and Finance. Each brief includes key message, likely objection, and 30-min meeting agenda.
// Deal Risk Assessment
Assesses technical, adoption, migration, organisational, and timeline risk. Produces a go/no-go recommendation and critical success factors.
// Executive Summary
Generates a 1-page CISO/CIO deliverable: key risks with business impact, 3-phase implementation roadmap, vendor recommendations, and next steps.
